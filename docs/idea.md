## 献立カレンダー  

10日分の献立を書き込めるカレンダーを作る

大枠は日付と朝、昼、夜の3列  
朝2列、昼と夜は3列  
各セルに食事を書き込める  

## 機能  

### セル  
通常時は文字を表示  
ダブルクリックすると編集できるテキストボックスに変化  
編集中のセル以外をクリックで編集確定  

日付以外のあらゆるセルが編集できる  

セルは100文字まで記入できる  
改行ができる  


### 日付  
10日分表示する  
今日が1番上の昇順  

`5月24日(日)`のようなフォーマット  

画面をリロードすると今日が一番上になる  

### 色  
土日行は背景色を薄いピンクにする  

### 保存  
30日分の献立を保存する  

献立以外のセルの情報も保存しておく  
献立以外は現在の値のみ保持  

過去の献立を表示する方法は今は考えない  

## 技術スタック  

### 開発環境  
bun vite react typescript
静的ビルドしてhtmlファイルを生成  

### ユーザー環境  
windows Edge
htmlファイルを常時タブに表示しておく  

端末数1  
他の端末で表示することは考えない  

### データ  
ブラウザに保存しておく  
バックアップは考えない
